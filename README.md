# R8 客室清掃品質 分析プロジェクト

稼働中の全ホテルの **R8年度（今年）集計表** をもとに、客室清掃の品質（クレーム率・口コミ清潔感指摘）を横断分析し、改善案までをまとめたプロジェクトです。

> ⚠️ **取扱注意**：本リポジトリは社内のクレームデータおよび担当者名（個人情報）を含みます。**非公開（private）** 前提で管理してください。

## 内容

| ファイル | 内容 |
|---|---|
| [`index.html`](./index.html) | 統合ランディング（成果物への入口） |
| [`dashboard.html`](./dashboard.html) | 全23ホテル 品質ダッシュボード（ランキング・品質マトリクス・ホテル別ドリルダウン） |
| [`report.html`](./report.html) | 改善提案レポート（件数インパクト起点の施策・ロードマップ） |
| [`data.json`](./data.json) | 全23ホテルの抽出データ（クレーム率・種別・担当別 等） |
| [`insights.json`](./insights.json) | レポート用に集計したインサイト |

いずれも単一HTMLで完結（外部依存なし）。ブラウザで `index.html` を開けば閲覧できます。

## データ範囲

- 対象：全23ホテル（通常稼働18／立ち上げ中4／データ未整備1）
- 期間：R8年度・概ね2〜7月（7月は現場により集計途中）
- 出典：Google Drive 各ホテル「R8_集計表」内「R8品質データまとめ」シート

## 主要な発見

- クレーム366件のうち **セット漏れ・残置・汚れの3種で64%** を占める。
- 口コミ（清潔感）指摘272件のうち **「臭い」が33.5%** で最大要因。クレーム率とは連動しない。
- クレーム率はほぼ全ホテルで前年（R7）比 **改善** 傾向。現在は「質の底上げ」フェーズ。

## 指標の定義

- **クレーム率** ＝ クレーム数 ÷ 清掃数
- **口コミ（清潔感）指摘率** ＝ 口コミ指摘数 ÷ 口コミ件数
  ※ クレーム率とは母数が異なるため直接比較はできない。

## 注記

担当者別の分析は、教育・支援・負荷平準化を目的としたものであり、評価や懲罰を意図するものではありません。
